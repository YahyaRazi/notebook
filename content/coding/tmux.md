## Install tpm

## Prefix

Prefix C-a means Ctrl+a as defined in [~/.tmux.conf](https://github.com/wayanjimmy/dotfiles/blob/master/.tmux.conf#L3)

Managing windows

```
C-a <             Show helper info
```

Finding window

```
<prefix> f find window
```
